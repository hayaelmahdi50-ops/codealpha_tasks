# Exploratory Data Analysis (EDA) Task

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the provided dataset to understand its structure, identify patterns, detect anomalies, and generate meaningful insights.

## Objectives

* Understand dataset structure and features
* Handle missing values and duplicates
* Detect outliers
* Analyze distributions of variables
* Explore relationships between features
* Summarize key findings

## Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Analysis Steps

1. Data Loading and Inspection
2. Data Cleaning
3. Missing Values Treatment
4. Duplicate Check
5. Outlier Detection
6. Univariate Analysis
8. Correlation Analysis
9. Final Insights

## Key Insights

* Some columns contained missing values and were handled appropriately.
* Outliers were detected in numerical features.
* Several relationships were identified between variables.
* The dataset shows useful trends that can support decision-making.

## Files Included
* `finalflightproject.ipynb` → Full analysis notebook
* images/ → Charts and visualizations
* tableau.twbx → Dashboard file


## How to Run
1. Open the notebook file.
2. Install required libraries.
3. Run all cells to reproduce the analysis.
